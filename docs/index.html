// Three Pillars

A self-improving agent organization

👁

Supervised

Real-time streaming cards show every tool call. Humans monitor agents from Feishu or Telegram on their phone. Access control via user/chat allowlists. You see everything agents do.

🧠

Self-Improving

MetaMemory is a shared knowledge store. Agents write what they learn, other agents retrieve it. The organization gets smarter every day without retraining any model.

🏭

Agent Organization

One command generates a full agent team. Agents delegate tasks to each other via the Agent Bus. Scheduled tasks run autonomously. Agents can even create new agents.

// Core Components

What's inside

SDK

Claude Code Kernel

Every bot is a full Claude Code instance — Read, Write, Edit, Bash, Glob, Grep, WebSearch, MCP, and more. Autonomous operation via bypassPermissions mode.

/skill

MetaSkill

Agent factory. /metaskill ios app generates a complete agent team (tech-lead + specialists + code-reviewer) after researching best practices.

MEM

MetaMemory

Embedded SQLite knowledge store with full-text search and Web UI. Agents read/write Markdown documents across sessions. Shared by all agents.

IM

IM Bridge

Chat with any agent from Feishu/Lark or Telegram, including mobile. Streaming cards with color-coded status and tool call tracking.

API

Agent Bus

REST API on port 9100. Agents delegate tasks to each other via curl. Create and remove bots at runtime. Exposed as the /metabot-api skill.

CRON

Task Scheduler

Agents schedule future work — "check back in 2 hours". Persists across restarts, auto-retries when busy.

// Get Started

One command to install

# Install MetaBot (interactive setup)
curl -fsSL https://raw.githubusercontent.com/xvirobotics/metabot/main/install.sh | bash

# Or manual install
git clone https://github.com/xvirobotics/metabot.git
cd metabot && npm install
cp bots.example.json bots.json && cp .env.example .env
npm run dev

Prerequisites: Node.js 18+, Claude Code CLI installed and authenticated.
